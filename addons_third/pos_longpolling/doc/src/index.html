Module provides following methods in Backend side:

  • Broadcasting a message to all opened POSes: self.env['pos.config].send_to_all_poses(channel_name, message)
  • Broadcasting a message to the POSes in pos_set : pos_set._send_to_channel(channel_name, message)
  • Sending a message to a specific POS: _send_to_channel_by_id(self, dbname, pos_id, channel_name)
POS will get notification only if it's subscribed to the specified channel_name.

Methods for the Client side:

  • Creating additional Bus to sync data with another Sync Server: add_bus(key, sync_server)
If you connect with your regular Odoo server, you shall not using add_bus
  • Subscribing to a specific channel: add_channel_callback: function(channel_name, callback, thisArg)
